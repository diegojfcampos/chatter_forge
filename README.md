## Run Locally

Clone the project

```bash
  git clone https://github.com/diegojfcampos/chatter_forge.git
```

Go to the project directory

```bash
  cd api_habit_helper
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
